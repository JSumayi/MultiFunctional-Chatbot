# Utility Chatbot

A simple, interactive chatbot built with Python and Streamlit.

## Features
- **Trip Planner**: Generate simple itineraries based on budget and duration.
- **Study Roadmap**: Get study guides for Python, DSA, and Web Development.
- **Weather Info**: Check current weather conditions (mock data).
- **News**: View top daily headlines.
- **General Chat**: Basic conversational abilities.

## Setup & Run

1.  Install requirements:
    ```bash
    pip install -r requirements.txt
    ```
2.  Run the application:
    ```bash
    streamlit run app.py
    ```
